# LocTubo

Projeto inicial do novo site institucional da LocTubo, criado com HTML, CSS e JavaScript puros.

## Objetivo

Modernizar a presenca digital da empresa e melhorar a captacao de clientes para pedidos de orcamento de equipamentos para construcao civil.

## Estrutura

- `index.html`: pagina principal com hero, catalogo por categoria, cards de produto, detalhes tecnicos, vantagens, processo e formulario.
- `css/styles.css`: tokens visuais, base responsiva e componentes reutilizaveis.
- `js/main.js`: menu mobile, carousel do hero, busca de equipamentos, preenchimento do formulario e envio para WhatsApp.
